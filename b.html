Problem Area

Usage of micro aerial vehicles for cleaning and inspection of high-rise structures with non-planar facades in a built environment.

Features

Unmanned Aerial Vehicle (Drone)

We implemented a semi-autonomous navigation algorithm for the UAV using several techniques.

Unmanned Ground Vehicle (Husky)

We implemented a motion coordinator to facilitate air-ground collaboration.

Simulation

We implemented a simulation environment for our use case of cleaning Supertrees.

User Interface

We implemented a user interface to serve as a control module to be used by operators.
